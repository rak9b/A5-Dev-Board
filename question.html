Question-1: What are the different ways to select an element in the DOM?

We can select DOM elements using methods like getElementById() for IDs, getElementsByClassName() for classes, and getElementsByTagName() for tags. Modern methods like querySelector() and querySelectorAll() use CSS selectors to find single or multiple elements efficiently.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML lets us get or change everything inside an element, including HTML tags. InnerText gives us only the visible text, ignoring hidden content. TextContent gives all the text, even if it’s hidden, but doesn’t recognize HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements. Instead of adding event listeners to multiple child elements, you add one listener to the parent, and it checks which child triggered the event.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM means that when an event happens on an element, it first runs on that element and then moves up to its parent, grandparent, and so on. For example, if you click a button inside a div, the click event first happens on the button, then on the div, and then on higher parent elements unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, we can create an element using document.createElement("tagName"). To add it to the page, we use appendChild() to place it inside a parent element or append() for multiple elements. To remove an element, we use removeChild() on its parent or simply call element.remove().
